University Project Presentation

Online Shopping System
Enhanced Edition

Java Swing based e-commerce desktop application with login, signup, product browsing, cart management, and checkout.

Presented in a clean academic style
TITLE01
Introduction

Project introduction

This project is an online shopping system developed in Java using Swing. It is designed to simulate a real shopping platform where users can sign up, log in, browse products, add items to cart, and place an order.

INTRODUCTION02
Problem Statement

Why this project is needed

  • To show how an e-commerce system works at a basic level.
  • To manage products, users, and cart items in an organized way.
  • To provide a simple but realistic shopping interface.
  • To practice object-oriented programming and GUI design.
MOTIVATION03
Objectives

Main objectives of the system

User authentication

Allow signup and login with customer details.

Product catalog

Show a large list of products with categories.

Shopping cart

Let users add, update, and remove products.

Checkout

Generate a final order receipt after purchase.

OBJECTIVES04
System Modules

Important components

Main.java

Application entry point and look-and-feel setup.

ShoppingSystem.java

Business logic, user management, and checkout.

ShoppingSystemGUI.java

Complete user interface and interaction layer.

ARCHITECTURE05
Data Model

Core classes used in the project

Product

Contains price, stock, discount, description, and image URL.

User

Stores name, email, password, phone, and address.

Cart

Handles selected items and total amount.

Order

Creates receipt and confirms the final purchase.

DATA MODEL06
Features

Main features of the application

  • Login and signup page for registered customers.
  • Guest browsing option for quick access.
  • 30 products across 6 categories.
  • Image cards with discount and stock display.
  • Search and category filter in the shopping area.
  • Cart, quantity update, and receipt generation.
FEATURE SET07
Working Flow

How the system works

Step 1

User opens landing page.

Step 2

Login, signup, or browse as guest.

Step 3

Search products and add to cart.

Step 4

Checkout and receive the order summary.

FLOW08
Advantages

Strength of this project

The application is simple to understand, visually attractive, and demonstrates practical software engineering concepts such as modular coding, object-oriented design, event handling, and user interaction.
